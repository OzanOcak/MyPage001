Game Programming

Hello everyone, my name is Ozan Ocak , I am primarily a software developer, game designer and 2D/3D computer graphic artist. My biggest hobby is learning something new. It can be a software tool or a programming language or even a spoken language.I do not think making a game is hard task to do. You just need to invest some time in learning some software programs and keep practicing about them.Let me talk about the process of 2D game development ; first you need some 2D graphics .You might find interesting what I do ,to create 2D graphics .I create 3D environments, 3D characters in Blender3D ,later I animate and render them to have some nice sprites. There are 2 reason behind of doing this . First I can have realistic 2D sprites , second is 3D design is really fun to learn and work with. After I upload all art images into Unity 3D game engine now time to animate character sprites and it is very easy compare to a few couple years ago. Unity doesn't require you any coding for animation except from getting data from her library so you don't need to deal with many boring and confusing arrays for animation.Finally here it is fun part, coding .... coding is not as difficult as you think .Programming is very similar to speaking languages ; at first you need to learn the structure of any languages in our case it is C#, then keep practicing , and writing other's codes until the day you can speak your own word or codes. I suggest you study someone's codes who really know how to program because the way you write is gonna be your accent and you don't want speak/write c# with a bad accent :)
